== Description ==

Social Way is a Wordpress Plugin that allows you to share, like or follow everyone and everything with a simple shortcode!
APIs and buttons with one click!
Instructions & How-to-use at http://www.cadienvanprojects.tk/cadien-lab/social-way-plugin/
You can easily insert shortcodes and widgets wherever you want in one click!
Add buttons automatically in every page and every post! You can list pages, buttons and icons you want to show!
With more than 40 shortcodes, you have the World in your hands!

== Installation ==

1. Upload the entire `socialwaybycp` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How many shortcodes are available in Social Way? =

Currently, there are 45 available shortcodes.

= Where can I use my shortcodes? =

You can use them wherever you want! With an improved management of the output code, you can insert them everywhere in the page!

= How many languages are available in this plugin? =

You can easily manage shortcodes for 250 languages with a single click!

= How many platforms does the plugin support? =

You have shortcodes for Facebook, Twitter, Google Plus, Spotify, Youtube, Tumblr, LinkedIn, Pinterest, Vimeo, Last.fm, Ask.fm, Skype & Steam.
